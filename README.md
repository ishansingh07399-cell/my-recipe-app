# 🍽️ Recipe Finder App

A beautiful and responsive React application that allows users to search for delicious recipes from around the world. Built with React and Vite, this app fetches real-time data from TheMealDB API to provide detailed ingredients, exact measurements, and step-by-step cooking instructions.

## ✨ Features

* **🔍 Smart Search:** Search for any recipe or main ingredient instantly.
* **📖 Recipe Details:** View high-quality images, comprehensive ingredient lists, and properly formatted cooking steps.
* **🔀 Dynamic Routing:** Seamless, fast navigation between search results and recipe pages using React Router.
* **📱 Clean UI/UX:** A modern, soft-themed user interface designed to reduce eye strain and provide a premium feel.
* **📺 Video Tutorials:** Direct links to YouTube cooking tutorials for supported recipes.
* **🛡️ Error Handling:** Graceful fallback UI when a user searches for a recipe that doesn't exist.

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Build Tool:** Vite
* **Routing:** React Router v6
* **Styling:** Custom CSS3
* **API:** [TheMealDB API](https://www.themealdb.com/api.php)

## 🚀 Getting Started (Run Locally)

To get a local copy up and running on your machine, follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your computer.

### Installation

1. Clone the repository:
   ```bash
   git clone [https://github.com/ishansingh0799-cell/my-recipe-app.git](https://github.com/ishansingh0799-cell/my-recipe-app.git)
Navigate into the project directory:

Bash
cd my-recipe-app
Install the required dependencies:

Bash
npm install
Start the local development server:

Bash
npm run dev
Open your browser and visit the local link provided in your terminal (usually http://localhost:5173).
